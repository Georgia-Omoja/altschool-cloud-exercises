LINUX COMMANDS

1. cal - displays the calender of the current month
1. free- displays the amount of free memory
1. file - determine the file type
1. info [command]- Display a command's info entry
1. ln - create hard and symbolic links.
1. uniq - report or omit repeated lines
1. grep (pattern) [file...] - Print lines matching a pattern
1. alias [command]- Create an alias for a command
1. wc [file...]- Print newline, word, and byte counts for each file
1. type [command] - Indicate how a command name is interpreted
